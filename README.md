This warehouse contains the code and data of the paper "Optimization Method for Open-Web Sandwich Panels Based on Physics-Informed Neural Networks"

Pinn.ipynb: the main implementation and training code of physical information neural network model.
Pso.ipynb: implementation code of particle swarm optimization algorithm.
B50g2000t200l400x100.xlsx and other.Xlsx files: model input parameters, boundary conditions, material properties or result data.
When training in the model, the address of the data needs to be modified accordingly
